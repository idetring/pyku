Climate Indicators at DWD
=========================

CLIX is a plugin developed to facilitate the calculation of climate indicators,
leveraging the robust capabilities of the **xclim library**. It adheres to the
specific definitions and methodologies for climate indicators as utilized at
the German Weather Service (DWD), while offering flexible parameter adjustment
to suit diverse analytical needs.

Designed with data interoperability in mind, CLIX strives to follow the `CF
Conventions <https://cfconventions.org/>`_ for output file generation, ensuring
that results are widely compatible and self-describing. Its versatile
architecture allows for seamless integration into other Python programs, or it
can be conveniently executed as a standalone application using the `clixrun`
command-line utility.

For detailed information on the xclim library, please refer to the `xclim
Official Documentation <https://xclim.readthedocs.io/en/stable/>`_.
